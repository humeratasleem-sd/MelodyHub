🎵 MelodyHub

MelodyHub is a lightweight and efficient music streaming application designed to provide a seamless audio listening experience. The platform focuses on simplicity, performance, and intuitive user interaction, enabling users to access and play music effortlessly.

📌 Overview

MelodyHub allows users to browse and play audio files stored locally within the application. It is built with a structured and scalable approach, making it suitable for both learning purposes and further enhancement into a full-featured music platform.

✨ Key Features
Smooth audio playback with essential controls (play, pause, skip)
Dynamic loading of songs from a centralized directory
Clean and responsive user interface
Efficient handling of local media files
Lightweight architecture with fast performance
📂 Media Management

All audio files are managed through a dedicated directory:

Important:
Every song must be placed inside the uploads/ folder.
The application is configured to fetch and play audio files exclusively from this location.

This ensures consistent file handling and simplifies the process of adding or updating songs.

🛠️ Technology Stack
Frontend: HTML5, CSS3, JavaScript
Architecture: Modular and file-based structure
(Optional backend integration can be extended using Node.js and Express)
⚙️ Setup Instructions

Clone the repository:

git clone https://github.com/humeratasleem-sd/MelodyHub.git

Navigate to the project directory:

cd MelodyHub
Add audio files:
Place all .mp3 (or supported formats) inside the uploads/ directory.
Run the application:
Open index.html in a browser or use a local development server.
